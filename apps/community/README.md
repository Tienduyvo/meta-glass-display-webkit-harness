# Community apps

Apps contributed by users. To use one: copy its `app.config.json` into `apps/<name>/` and add a
line to `apps/registry.json` (or run `python tools/new_app.py` and paste the fields).

To contribute your own: open a pull request adding `apps/community/<slug>/app.config.json`
(see [`CONTRIBUTING.md`](../../CONTRIBUTING.md)), or use the **“Submit an app”** issue template,
or just send someone a **Share link** from the launcher. **No secrets/URLs/tokens in configs.**
