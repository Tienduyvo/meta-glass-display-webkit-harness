# CLAUDE.md

This repository is **agent‑driven**. Follow **[`AGENTS.md`](AGENTS.md)** — it is the setup‑wizard
and app‑builder runbook for this project.

Start every session by running `python tools/status.py`, tell the user where they are, and guide
them to the single next step (set up backend → build/change an app → publish). Don't dump all
steps at once. Never commit secrets or personal data.
